# ludum-dare-42

The game I made at [CampJS IX](https://ix.campjs.com/) for [Ludum Dare 42](https://ldjam.com/events/ludum-dare/42/).

## Getting started

```sh
npm install
npm start
```

Then open [http://localhost:8000/](http://localhost:8000/)
